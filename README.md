# CLI-Habit-Tracker-App
A command-line habit tracker built in Python to help users track daily habits and monitor progress over time.

# What it does:
A command-line habit tracking application built in Python that allows users to create habits, track daily completion, and view progress statistics such as streaks and completion rates.

## Features
- Add new habits
- Delete existing habits
- Mark habits as completed
- View streaks and completion rates
- Persistent storage using JSON

## Why I Built This
I built this to practice Python fundamentals such as file handling, data structures, and error handling, while solving a real-world problem of habit tracking.

## Tech Stack
 - Python
 - JSON (data storage)

## How to Use
1. Clone the repository
2. Run `main.py` in your terminal
3. Follow the on-screen prompts to manage your habits (By using the menu options to add, mark as completed, view stats, or delete habits)
4. Your habits and progress will be saved in `habits.json` for future reference.
